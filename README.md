# CryoCell: Cold-Environment Battery Simulator
A software simulator evaluating lithium-ion thermal degradation in sub-zero aerospace environments.

## Overview
This project is a discrete-time software simulation designed to model the thermal degradation and voltage sag of lithium-ion power systems operating in extreme cold environments (-40°C), specifically for More Electric Aircraft (MEA) applications.

## The Engineering Problem
As aerospace shifts toward hybrid-electric propulsion, cold-weather battery efficiency is a critical failure point. This simulator uses an equivalent circuit model and lumped capacitance thermal modeling to calculate the trade-off between self-heating power consumption and usable motor output.

## Tech Stack & Methodology
Language: Python, C++, MATLAB

Physics Modeled: Arrhenius equation for internal resistance, Joule heating, and state-of-charge (SOC) Coulomb counting.
